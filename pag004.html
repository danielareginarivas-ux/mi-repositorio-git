Pagina de proyectos en Roblox Studio

Roblox Studio te permite crear juegos usando el lenguaje de programación Lua. Es una excelente herramienta para aprender programación y desarrollo de videojuegos.

Exemplo de Script Lua


script.Parent.Touched:Connect(function(hit)

local character = hit.Parent

while character and not character:FindFirstChild("Humanoid") do
character = character.Parent
end

if character then
print("Jogador tocou o checkpoint!")
end
script.Parent.Color = Color3.fromRGB(0, 255, 0)
script.Parent.Material = Enum.Material.Neon
script.Parent.Touched:Connect(function(hit)

local humanoid = hit.Parent:FindFirstChildOfClass("Humanoid")

if humanoid then
script.Parent.BrickColor = BrickColor.new("Lime green")
print("Checkpoint alcançado!")
end
while true do
script.Parent.Orientation =
script.Parent.Orientation + Vector3.new(0, 1, 0)

wait(0.03)
end
end)
end)
