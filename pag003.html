Este proyecto se desarrolló en Visual Basic para automatizar el cálculo de los promedios de los estudiantes en una hoja de cálculo de Excel.
El programa recorre una lista de estudiantes, suma sus calificaciones, calcula el promedio final y determina automáticamente si cada estudiante aprobó o reprobó. Para realizar esta tarea, el código utiliza estructuras de repetición (For...Next) para iterar por las filas y columnas de la hoja de cálculo, almacenando y procesando los valores de las calificaciones. Después de calcular el promedio de cada estudiante, una estructura condicional (If...Else) verifica si el promedio es mayor o igual a 6. Si es así, el estudiante recibe el estado "Aprobado"; de lo contrario, recibe el estado "Reprobado". Este proyecto demuestra conceptos importantes de programación, como variables, bucles de repetición, condicionales, operaciones matemáticas y manipulación de datos en hojas de cálculo, lo que hace que el proceso de evaluación sea más rápido y eficiente. Sub CalcularPromedioYEstado()
Dim i As Integer ' Variable para el bucle de alumnos
Dim j As Integer ' Variable para el bucle de notas
Dim sumaNotas As Double ' Variable para almacenar la suma de las notas
Dim promedio As Double ' Variable para el promedio de las notas
' Bucle para recorrer los alumnos
For i = 3 To 7 ' Fila 3 a fila 7 (suponiendo que los alumnos están en las
filas 3 a 7)
' Inicializamos la suma de notas
sumaNotas = 0
' Bucle para sumar las notas
For j = 2 To 5 ' Columna 2 a columna 5 (suponiendo que las notas están
en las columnas B a E)
sumaNotas = sumaNotas + Cells(i, j).Value
Next j
' Calculamos el promedio
promedio = sumaNotas / 4
' Determinamos si aprobó o reprobó
If promedio >= 6 Then
Cells(i, 6).Value = "Aprobado"
Else
Cells(i, 6).Value = "Reprobado"
End If
Next i
End Sub
